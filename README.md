# Windows-Update
As a security risk measure, I demonstrated how to update the Windows operating system with patches that are created to fix bugs, protect against new attacks, address known vulnerabilities, and add new features.

---------------------------------------------

1. Go to settings and select Update & Security and from there look for the "Check for updates" and select it. You can also check the history of your computer updates
